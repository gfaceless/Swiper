Swiper3-hdx (Swiper3.0.7)
==========

说明
-------
`swpier-hdx`对原版最主要的改动是内置`push`效果

API:
-------
传递的默认参数有变化，请根据自己需要更改:
```javascript
direction: 'vertical' // "horizontal"
speed: 200 // 300
effect: "push" // "slide"
longSwipesRatio: 0.33, // 0.5
```
